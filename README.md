Tech stack used:

* Html
* CSS
* JavaScript
* React JS
* Material UI
* TypeScript

# Features

* Play/Pause Button for auto sliding.
* Prev and next button for changing Slides.
* On clicking Thubmnail displays Details.
* Active Slide highlighted.
